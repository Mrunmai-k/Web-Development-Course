git init = initialize the current directory as git dir
--------------------------------------------------------------------------------------------------------
git add . = add everything in the current dir
git status = to check the status
git log = to get the log of commit history
--------------------------------------------------------------------------------------------------------
Before adding to the staging area:
git diff <filename> = to check the diff betn previous and current version of the specified file.
git checkout <filename> = checkout to previous commit 
--------------------------------------------------------------------------------------------------------
git remote add origin <URL> = adds the url of remote repo     
git push -u <remote name> <branch name> = push the local repo to remote repo
--------------------------------------------------------------------------------------------------------
git branch <branch_name> = create a new branch 
git branch = returns the list of branches
git checkout <branchname> = changes to specified branch
git merge <other_branch> = merges the other branch in main/master
